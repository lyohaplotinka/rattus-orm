`@rattus-orm/solidjs` это Data Provider, связывающий [Сигналы Solid.js](https://www.solidjs.com/tutorial/introduction_signals) с Rattus ORM.

В пакет выходит:
1. SolidjsDataProvider
2. хуки `useRattusContext`, `useRepository`
3. компонент `<RattusProvier />`.
