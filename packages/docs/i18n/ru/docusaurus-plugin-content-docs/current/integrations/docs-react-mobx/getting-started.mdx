`@rattus-orm/react-mobx` это Data Provider, связывающий [React MobX](https://mobx.js.org/README.html) с Rattus ORM.

В пакет выходит:
1. ReactSignalsDataProvider
2. хуки `useRattusContext`, `useRepository`
3. компонент `<RattusProvier />`.
