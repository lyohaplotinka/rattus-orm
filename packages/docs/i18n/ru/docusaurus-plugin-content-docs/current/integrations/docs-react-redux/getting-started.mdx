`@rattus-orm/react-redux` это Data Provider, связывающий [React Redux](https://redux.js.org/) с Rattus ORM.

В пакет выходит:
1. ReactReduxDataProvider
2. хуки `useRattusContext`, `useRepository`
3. компонент `<RattusProvier />`.
