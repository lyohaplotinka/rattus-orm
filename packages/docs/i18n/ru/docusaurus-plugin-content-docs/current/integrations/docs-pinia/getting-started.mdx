`@rattus-orm/pinia` – это Data Provider, связывающий хранилище
[Pinia](https://pinia.vuejs.org/) с Rattus ORM.

В пакет входит: 
1. PiniaDataProvider
2. композиции `useRattusContext`, `useRepository`
3. плагин для Vue, настраивающий связь ORM с Pinia.
