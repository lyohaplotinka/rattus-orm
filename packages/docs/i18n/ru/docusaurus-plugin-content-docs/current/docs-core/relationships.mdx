# Связи

Все связи между моделями полностью копируют связи из
`@vuex-orm/core@next`. Чтобы разобраться, как
организованы связи, рекомендуем вам ознакомиться с
документацией по связям из Vuex ORM Next.

[Читать документацию](https://next.vuex-orm.org/guide/relationships/getting-started.html)

:::info[Инфо]
Помните, что Rattus ORM более ориентирован на использование
с TypeScript. Для документации по декораторам связей
вы можете ознакомиться [с этим разделом](https://next.vuex-orm.org/guide/model/decorators.html).
:::
