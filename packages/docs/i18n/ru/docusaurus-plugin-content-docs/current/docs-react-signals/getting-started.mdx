`@rattus-orm/react-signals` это Data Provider, связывающий [React Signals](https://github.com/preactjs/signals/tree/main/packages/react) с Rattus ORM.

В пакет выходит:
1. ReactSignalsDataProvider
2. хуки `useRattusContext`, `useRepository`, `useRepositoryComputed`
3. компонент `<RattusProvier />`.
