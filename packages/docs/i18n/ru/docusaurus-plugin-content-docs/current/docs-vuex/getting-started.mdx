`@rattus-orm/vuex` – это Data Provider, связывающий хранилище
[Vuex](https://vuex.vuejs.org/) с Rattus ORM.

В пакет входит: 
1. VuexDataProvider
2. композиции `useRepository`, `useRepositoryComputed`
3. плагин для Vuex.
