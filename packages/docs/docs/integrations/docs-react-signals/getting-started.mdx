`@rattus-orm/react-signals` is a Data Provider that connects the [React Signals](https://github.com/preactjs/signals/tree/main/packages/react) with Rattus ORM.

The package includes:
1. ReactSignalsDataProvider
2. hooks `useRattusContext`, `useRepository`
3. `<RattusProvier />` component.
