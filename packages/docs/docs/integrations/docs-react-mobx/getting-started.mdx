`@rattus-orm/react-mobx` is a Data Provider that connects the [React MobX](https://mobx.js.org/README.html) with Rattus ORM.

The package includes:
1. ReactMobxDataProvider
2. hooks `useRattusContext`, `useRepository`
3. `<RattusProvier />` component.
