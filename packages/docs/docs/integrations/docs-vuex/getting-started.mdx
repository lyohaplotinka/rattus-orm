`@rattus-orm/vuex` is a Data Provider that links the [Vuex](https://vuex.vuejs.org/) store with Rattus ORM.

The package includes:
1. VuexDataProvider
2. compositions `useRattusContext`, `useRepository`, `useRepositoryComputed`
3. a plugin for Vuex.
