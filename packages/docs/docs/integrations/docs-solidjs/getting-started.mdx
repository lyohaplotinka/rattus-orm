`@rattus-orm/solidjs` is a Data Provider that links the [Solid.js Signals](https://www.solidjs.com/tutorial/introduction_signals) with Rattus ORM.

The package includes:
1. SolidjsDataProvider
2. hooks `useRattusContext`, `useRepository`
3. the component `<RattusProvider />`.
