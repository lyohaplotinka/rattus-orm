`@rattus-orm/react-redux` is a Data Provider that connects [React Redux](https://redux.js.org/) with Rattus ORM.

The package includes:
1. ReactReduxDataProvider
2. hooks `useRattusContext`, `useRepository`
3. the component `<RattusProvider />`.
